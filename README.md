## 📂 Proyectos destacados

### 🔐 SSH Brute Force Attack (Hydra)

Simulación de ataque de fuerza bruta contra SSH en entorno controlado.

Incluye todo el ciclo de un incidente de seguridad:

* Ejecución de ataque con Hydra
* Detección en logs del sistema (`/var/log/auth.log`)
* Análisis de tráfico TCP (SYN / SYN-ACK)
* Mitigación mediante iptables
* Verificación del impacto

👉 Proyecto disponible en este repositorio:  
[DualLayer-Security/SOC-Lab-SSH-BruteForce](https://github.com/DualLayer-Security/SOC-Lab-SSH-BruteForce)
